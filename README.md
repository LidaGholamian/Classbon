# Classbon LMS

A Learning Management System (LMS) built with Next.js and React.

This project was developed as a large-scale educational project to simulate a production-ready learning platform and improve skills in modern frontend development.

---

## Overview

Classbon is an online learning platform that allows users to browse courses, authenticate using mobile verification, and access educational content.

The project focuses on building scalable frontend architecture using modern React and Next.js patterns.

## Challenges Solved

During development, several real-world frontend challenges were addressed:

### Authentication Flow

Implemented mobile authentication and verification code workflow.

### Notification System

Built a reusable notification system using Zustand.

### Timer Component

Developed a reusable countdown timer component using:

- forwardRef
- useImperativeHandle
- react-timer-hook

### Auth Code Component

Created a custom OTP input component with:

- Auto focus
- Keyboard navigation
- Backspace handling
- Imperative methods

### API Error Handling

Implemented centralized API error handling using:

- Axios Interceptors
- React Query Mutation Cache
- Global Notifications
